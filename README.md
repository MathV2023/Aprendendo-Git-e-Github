# Jogo: Adivinhe o número

**Obs.:** Este é um jogo criado em lua para eu aprender como usar o git.

## Mecânicas:
- O jogo irá escolher um número aleátorio que é limitado a ser 1, 100 ou outro número natural entre eles;
- É dada 7 tentativas para acertar o número, se acabar as todas as suas chances você perdeu, já se tu tiver acertado o número gerado pelo jogo, você vence o ele.
